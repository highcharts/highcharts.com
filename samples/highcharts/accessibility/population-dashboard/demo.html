Highcharts Population Dashboard

This is an example dashboard highlighting some accessibility features of Highcharts. The dashboard visualizes population data for 7 Nordic countries, and involves multiple interactive charts. The chart data is collected from WorldBank.org, using census data from 1984 to 2019.

Clicking on countries in the map will display population details for that country in the adjacent charts.
